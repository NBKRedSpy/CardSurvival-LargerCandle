# Larger Candle

A candle which is three times as large as a regular candle.  

The large candle takes the same amount of resources as the normal candles (which produce three candles); however, a large candle will burn three times as long and is three times as heavy.

DeepL translated:
一个比普通蜡烛大三倍的蜡烛。 

大蜡烛需要的资源与普通蜡烛（产生三支蜡烛）相同；但是，大蜡烛的燃烧时间是普通蜡烛的三倍，重量也是普通蜡烛的三倍。

# Install

## Mod Manager
Use the NexusMod's Vortex Mod manager to install.

## Manual Install
Requires the CSTI-Modloader to be installed

https://www.nexusmods.com/cardsurvivaltropicalisland/mods/23

Extract this mod's zip file into the root of the plugins folder.
